- 🤖 **AI Automations**: Expert in **n8n** (self-hosted & cloud), Zapier, LangChain, and API-first AI integrations.

- 📦 **Databases & Cloud**: Proficient in MongoDB, MySQL, Firebase, MS SQL Server, and AWS.
